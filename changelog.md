# Changelog

## v1.0.3

- Internal/infrastructure changes only

## v1.0.2

- Add support for `//`, `#`, and `/* */` comment styles
- Make commentStyle option configurable as a passable prop

## v1.0.0

- Add `taglifyText` function to replace content between HTML comment markers
- Add `taglifyFile` function to apply tag replacements to files
- Add TypeScript definitions for improved type support
- Improve error handling with descriptive messages for missing files
